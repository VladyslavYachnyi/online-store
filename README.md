# Online Store (Spring Boot)

Simple online store web application built with Spring Boot as a learning project.

## Features
- User registration and login
- MVC architecture
- Thymeleaf-based HTML pages
- Layered backend architecture
- REST-ready backend structure

## Technologies
- Java
- Spring Boot
- Spring MVC
- Spring Data JPA
- Thymeleaf
- Maven
- SQL database

## Project Structure
- `controller` – web controllers
- `service` – business logic
- `repository` – data access layer
- `model` – entities
- `templates` – Thymeleaf HTML templates

## Run

### Run using IntelliJ IDEA
1. Open the project in IntelliJ IDEA
2. Run the `OnlineStoreApplication` class
3. Open in browser: `http://localhost:8080`

### Run using Maven
```bash
mvn spring-boot:run
Then open in browser: http://localhost:8080
